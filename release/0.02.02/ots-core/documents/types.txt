Wat hebben we in eerste instantie verder nodig?
- Scalar Abs Float Value
- Scalar Rel Float Value
- Scalar Abs Double Value
- Scalar Rel Double Value
- Vector Abs Float Dense
- Vector Rel Float Dense
- Vector Abs Double Dense
- Vector Rel Double Dense
- Matrix Abs Float Dense
- Matrix Rel Float Dense
- Matrix Abs Double Dense
- Matrix Rel Double Dense


Wat mag:
abs-abs => rel
abs+rel => abs
abs-rel => abs
rel+rel => rel
rel-rel => rel
rel+abs => abs
rel*c => rel

Wat mag niet:
abs+abs
rel-abs
abs*c
