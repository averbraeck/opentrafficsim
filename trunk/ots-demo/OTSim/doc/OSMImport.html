Creating an OSM file

  1. Navigate with your web browser to http://www.openstreetmap.org.
  2. Zoom and pan to put the area that you want to export in view.
  3. Click on the Export button (along the top of the window).
  4. Edit the WGS84 coordinates (if needed).
  5. Click the Export button and save the resulting .osm file in a convenient location.

Importing an OSM file

  1. Start OTSIM (do not load a model or network).
  2. Click on the menu File, then Import OpenStreetMap Network.
  3. Navigate to the .osm file to load, select it and click the Open button.
